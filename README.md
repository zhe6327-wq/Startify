# Startify
A task-starting web app that helps users overcome procrastination through task recommendations, countdowns, and custom task creation.
Startify 是一款创新的效率工具，专为解决“知道要做，却难以开始”的拖延困境而设计。我们不只是提供一个倒计时器，而是通过 AI 智能将令人望而生畏的大任务，拆解成一个“此刻就能开始的小动作”。 核心理念：先启动，再进入状态。 设计四大核心页面，通过状态匹配推荐、极简倒计时、任务清单与自定义创建，帮助用户快速打破拖延，迈出行动第一步。
